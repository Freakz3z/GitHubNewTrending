## rebane2001/x86CSS

![GitHub stars](https://img.shields.io/badge/stars-290-yellow)

一个完全使用 CSS 构建的 x86 模拟器。无需任何 JavaScript，仅通过层叠样式表（CSS）的样式选择器和动画特性实现了一个可运行的 8086 CPU。项目支持运行编译自 GCC 的 C 程序机器码，通过 CSS 的动画和容器查询机制驱动 CPU 运行，是一个极具创意的技术探索项目。

- **技术栈**: HTML/CSS, Python (构建脚本)
- **链接**: [rebane2001/x86CSS](https://github.com/rebane2001/x86CSS)

## lifo-sh/lifo

![GitHub stars](https://img.shields.io/badge/stars-229-yellow)

一个浏览器原生操作系统，重新构想了 Unix/Linux 的设计理念。浏览器运行时充当内核，浏览器 API 作为系统调用。实现了 60+ 个 Unix 命令、类 bash shell、虚拟文件系统以及 IndexedDB 持久化。核心特性包括 POSIX 兼容的虚拟文件系统（支持硬链接、符号链接、权限）、Node.js 兼容层、以及完整的命令注册系统。

- **技术栈**: TypeScript, xterm.js, IndexedDB, Web APIs
- **链接**: [lifo-sh/lifo](https://github.com/lifo-sh/lifo)
